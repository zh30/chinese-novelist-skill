# 叙事节奏曲线设计

## 背景

当前 skill 缺乏叙事节奏曲线的设计机制。用户知道好的小说紧张程度应该有起伏，但：
- 不知道怎么设计曲线
- 不知道关键节点在哪
- 写着写着曲线就平了
- 写完后不知道怎么验证

## 目标

创建叙事节奏曲线系统：
1. **SKILL.md** — 增加节奏曲线设计章节
2. **模板** `07-叙事节奏曲线.md` — 曲线设计、节点定义、跟踪验证

## 设计方案

### 1. 叙事节奏曲线类型

预设几种常见的节奏曲线模板，用户选择后填充：

**A. 递进型**
```
紧张度：
★★★★★                    ★★★★★
      ★★★★            ★★★★
          ★★★      ★★★
              ★★★★
                  ★★★★★
第1章 →        中段        → 结局
```
适用：升级流、复仇流

**B. 波浪型**
```
紧张度：
★★★★        ★★★★
    ★★★          ★★★
        ★★      ★★
            ★★★★
第1章 →    中段1  中段2  → 结局
```
适用：悬疑、探险

**C. 高原型**
```
紧张度：
★★★★★  ★★★★  ★★★★
    ★★★      ★★★
        ★★★
第1章 →        中段        → 结局
```
适用：都市、职场

**D. 反转型**
```
紧张度：
              ★★★★★
★★★★★      ★★★
    ★★★★
        ★★★
第1章 →        中段        → 结局
```
适用：宫斗、权谋

### 2. 关键节点定义

| 节点 | 位置 | 紧张度要求 | 必须元素 |
|------|------|-----------|----------|
| 开篇钩子 | 第1章 | 中高 | 危机/悬念/目标 |
| 第一个小高潮 | 前10% | 中高 | 小胜利或小失败 |
| 中点转折 | 50% | 最高 | 认知翻转/重大失败 |
| 中段升级 | 60-70% | 高 | 代价增大 |
| 倒数第二个高潮 | 85% | 很高 | 终极选择 |
| 结局 | 100% | 最高 | 终极对决/终极选择 |

### 3. 曲线跟踪机制

每写完5章，记录一次紧张度评分（1-5星），对照设计曲线检查。
