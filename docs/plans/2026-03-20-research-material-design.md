# 素材积累阶段设计

## 背景

当前 workflow 跳过了素材积累阶段，用户可能在还没研究清楚领域知识的情况下就开始写作，导致：

- 写出来的内容缺乏真实感
- 专业人士一眼看出破绽
- 关键细节缺失，影响沉浸感

## 目标

在工作流中增加"素材积累"阶段，包含：
1. **SKILL.md** — 增加素材积累阶段说明
2. **模板** `04-素材积累.md` — 素材清单和整理方式

## 设计方案

### 1. SKILL.md 增加素材积累阶段

在"策划模式"和"试写模式"之间，增加"素材积累阶段"的说明。

**触发条件：**
- 用户写的题材涉及专业领域（医疗、法律、历史、职场等）
- 用户不确定自己是否了解足够的背景知识
- 用户主动要求"做功课"

**素材积累阶段流程：**

```
第一步：确定素材范围
- 根据题材类型，列出需要研究的领域
- 参考 04-素材积累.md 的题材素材清单

第二步：收集素材
- 网络搜索、阅读相关书籍
- 采访相关从业者（如适用）
- 实地考察（如适用）

第三步：整理素材
- 按模板结构整理到 04-素材积累.md
- 标记"必须准确"vs"可以虚构"的内容

第四步：验证素材
- 检查关键细节是否准确
- 确保没有常识性错误
- 标注敏感内容（如有）
```

### 2. 素材积累模板

**文件：** `novels/<书名>/04-素材积累.md`

**用途：** 整理和存储研究所收集的素材。

```markdown
# 素材积累

> 本文件用于整理写作所需的领域知识。写完记得更新。

## 素材清单

### 必查项目（根据题材）

- [ ] 领域基础知识（术语、行话、基本规则）
- [ ] 从业者日常（工作流程、挑战、习惯）
- [ ] 常见错误（外行人容易犯的错）
- [ ] 专业细节（能增加真实感的小细节）

### 题材素材清单

**职场/商战**
- [ ] 行业术语和缩写
- [ ] 典型工作日流程
- [ ] 会议/汇报的语言风格
- [ ] 常见职业病/压力源
- [ ] 行业潜规则

**医疗**
- [ ] 医院科室分工
- [ ] 常见诊断流程
- [ ] 医疗术语（患者能理解的）
- [ ] 医患沟通话术
- [ ] 医疗行业的禁忌话题

**历史**
- [ ] 当时日常用语
- [ ] 饮食习惯、服饰、交通
- [ ] 社会阶层和礼仪
- [ ] 货币/物价/工资
- [ ] 常见历史错误（避免穿越）

**悬疑/犯罪**
- [ ] 侦查基本流程
- [ ] 常见取证方法
- [ ] 犯罪心理学基础
- [ ] 法律程序（拘留、审讯、起诉）
- [ ] 警察/法医真实工作状态

**科幻**
- [ ] 科技术语（用普通人能理解的方式解释）
- [ ] 技术现状和局限
- [ ] 逻辑自洽性检查
- [ ] 科学顾问（如需要）

### 自定义领域

根据你的具体题材，列出需要研究的领域：

#### [领域名称]

**必查项：**
- ...

**加分项（增加真实感）：**
- ...

**常见错误：**
- ...
```
