# 叙事节奏框架（三层体系）

> 从"只能事后检查"到"事前可规划"的节奏设计系统。

---

## 三层架构概览

```
┌─────────────────────────────────────┐
│  宏观层（全本视角）                   │
│  • 四幕结构 + 10-15个关键节点         │
│  • 适用于：大纲规划、整体结构把控      │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│  中观层（章节组视角）                 │
│  • 每3章为一个"节奏单元"              │
│  • 适用于：连载规划、追更节奏控制      │
└──────────────┬──────────────────────┘
               │
┌──────────────▼──────────────────────┐
│  微观层（单章视角）                   │
│  • 每章内部3次情绪起伏               │
│  • 适用于：每日写作的具体执行          │
└─────────────────────────────────────┘
```

## 第一层：宏观节奏（全本规划）

### 四幕结构

```
第一幕（设局）    第二幕（对抗）    第三幕（高潮）    第四幕（结局）
[====25%====]    [=====50%=====]    [====20%====]    [====5%====]
     │                  │                  │               │
   建立冲突          升级对抗           最终对决          收尾收束
   引入世界          代价增大           认知翻转          情感落点
```

| 节点 | 位置 | 功能 | 紧张度 | 悬念强度 |
|------|------|------|--------|---------|

| 3.第一情节点 | 10% | 主角接受挑战 | ★★★☆☆ | 3级 |

| 6.趣味和游戏 | 30% | 探索新世界 | ★★☆☆☆ | 2级 |
| 7.中点转折 | 50% | 虚假胜利/失败 | ★★★★★ | 5级 |

### 宏观节奏检查

每完成10章，对照以下检查：
- [ ] 当前进度与节点规划是否匹配
- [ ] 紧张度曲线是否呈上升趋势（允许波动）
- [ ] 是否有连续5章紧张度低于2星（过平警告）

## 第二层：中观节奏（章节组规划）

### 黄金三章模式

每3章组成一个"节奏单元"，形成小闭环：

```
第N章        第N+1章      第N+2章
[铺垫]  →  [升级]   →  [小高潮+钩子]
  │            │            │
建立情境    矛盾激化    情绪高点+新问题
```

### 节奏单元类型

#### 类型A：推进型
```
第1章：主角发现问题 → 第2章：尝试解决失败 → 第3章：找到新线索（小胜利）
紧张度：★★☆ → ★★★ → ★★★☆
```

#### 类型B：转折型
```
第1章：平静日常 → 第2章：突发事件 → 第3章：被迫改变（小失败）
紧张度：★☆☆ → ★★★ → ★★★☆
```

#### 类型C：情感型
```
第1章：误会/心动 → 第2章：互动加深 → 第3章：阻碍出现（关系变化）
紧张度：★★☆ → ★★★ → ★★★☆
```

### 中观节奏检查

每3章（一个单元）完成后：
- [ ] 第3章结尾是否有强钩子
- [ ] 3章内是否有情绪起伏（低→高）
- [ ] 单元结束后是否推进了主线

## 第三层：微观节奏（单章设计）

### 单章情绪起伏

每章内部至少3次情绪变化：

```
20%位置        50%位置        80%位置
进入冲突  →   矛盾激化   →   钩子落下
  起点          中点          终点
(紧张↑)       (紧张↑↑)      (紧张↑↑↑)
```

### 章节内部结构

| 位置 | 功能 | 内容 | 字数占比 |
|------|------|------|---------|
| 0-20% | 进入点 | 尽快进入冲突/异常 | 20% |
| 20-50% | 施压点 | 阻碍升级，信息差 | 30% |
| 50-80% | 转折点 | 线索反转/关系变化 | 30% |
| 80-100% | 钩子点 | 悬念落下 | 20% |

### 微观节奏检查

每章交付前：
- [ ] 20%位置是否已进入冲突
- [ ] 50%位置是否有明确转折
- [ ] 80%位置是否有强钩子
- [ ] 字数是否达标（≥3000字）

## 题材特定节奏模板

### 模板A：悬疑/推理

**宏观节奏**：波浪型（紧张度波浪起伏）

**中观单元**（每3章）：
```
疑问 → 调查 → 线索/反转
[发现尸体] → [走访调查] → [新嫌疑人出现]
紧张度：★★★ → ★★☆ → ★★★☆
```

**微观节奏**（单章）：
```
20%：异常发现（尸体/密室）
50%：线索冲突（不同证词矛盾）
80%：钩子落下（新嫌疑人/新证据）
```

**节奏特点**：
- 第3章结尾必须有案件进展
- 每5-7章一个小高潮（新嫌疑人/排除嫌疑人）
- 中点（50%）必须有大反转（嫌疑人是凶手/凶手另有其人）

---

### 模板B：玄幻升级流

**宏观节奏**：递进型（紧张度持续上升）

**中观单元**（每3章）：
```
挑战 → 修炼 → 突破
[强敌出现] → [闭关修炼] → [突破境界]
紧张度：★★☆ → ★★☆ → ★★★☆
```

**微观节奏**（单章）：
```
20%：遭遇强敌/困境
50%：突破/获得新能力
80%：震惊众人/新挑战出现
```

**节奏特点**：
- 每3章一次小升级
- 每10章一次大升级（境界突破）
- 升级后必须有"震惊众人"的爽点
- 中点（50%）获得关键能力/功法

---

### 模板C：言情/情感

**宏观节奏**：波浪型（情感起伏）

**中观单元**（每3章）：
```
误会/心动 → 甜蜜 → 阻碍/危机
[初次心动] → [甜蜜互动] → [误会/情敌出现]
紧张度：★★☆ → ★★★ → ★★★☆
```

**微观节奏**（单章）：
```
20%：情感触发点（对视/对话）
50%：关系推进（牵手/告白）
80%：阻碍出现（误会/分离）
```

**节奏特点**：
- 甜虐交替，不能一直甜或一直虐
- 每3章一次关系变化（升温→阻碍→和解）
- 中点（50%）关系确定/重大误会
- 后30%解决外部阻碍（家庭/身份/误会）

---

### 模板D：种田/基建

**宏观节奏**：渐进型（目标逐步达成）

**中观单元**（每3章）：
```
目标 → 阻碍 → 解决+收获
[确定目标] → [资源不足] → [解决+新目标]
紧张度：★★☆ → ★★☆ → ★★★☆
```

**微观节奏**（单章）：
```
20%：遇到具体问题
50%：解决过程（技术/资源/人手）
80%：解决+新目标/新问题
```

**节奏特点**：
- 成就感是核心驱动力
- 每章都有小收获（种菜/建房子/招人手）
- 每3章一次大收获（收成/建成/扩张）
- 收获后必须出现新挑战（天灾/盗匪/市场波动）

---

### 模板E：都市现实

**宏观节奏**：高原型（紧张度保持中高水平）

**中观单元**（每3章）：
```
压力 → 应对 → 暂时解决+新压力
[职场危机] → [努力应对] → [暂时解决+更大危机]
紧张度：★★★ → ★★★ → ★★★☆
```

**微观节奏**（单章）：
```
20%：现实压力（工作/家庭/健康）
50%：应对过程（选择/牺牲）
80%：结果+新压力
```

**节奏特点**：
- 压力来源要具体（上司/房贷/父母）
- 主角应对要真实（不是开挂，是挣扎+成长）
- 中点（50%）价值观被挑战
- 结局不一定是圆满，但要有成长

## 节奏健康度量化

### 五维评分

每章自评（1-5星）：

| 维度 | 1星 | 3星 | 5星 |
|------|-----|-----|-----|
| **紧张度** | 过于平淡 | 有起伏 | 张力十足 |
| **情绪类型** | 单一无聊 | 有变化 | 丰富多样 |
| **钩子强度** | 无钩子 | 一般 | 迫切想看下章 |
| **信息量** | 无新信息 | 少量 | 重要揭示 |
| **期待值** | 不想再看 | 可有可无 | 迫不及待 |

### 健康标准

- **红灯**：连续3章总分<10分（需要大幅修改）
- **黄灯**：单章某维度1-2星（需要优化）
- **绿灯**：每章总分≥12分（良好）

### 节奏曲线可视化

在进度仪表盘中自动绘制：

```
紧张度曲线（最近10章）

★★★★★    ╱╲      ╱╲
★★★★☆   ╱  ╲    ╱  ╲___
★★★☆☆  ╱    ╲__╱       ╲__
★★☆☆☆ ╱                      ╲
★☆☆☆☆╱                        ╲_
      第1章  第3章  第5章  第7章  第9章

预警：第6-7章紧张度过低，建议加强
```

## 喘息机制（情绪低谷设计）

### 什么是喘息章

不是"无聊章"，而是**信息整合+人物关系深化**的章节。

### 喘息章结构

```
[20%] 上一事件的余波（简短回顾，≤500字）
[60%] 收获盘点 / 人物关系推进 / 世界观揭示
[20%] 新威胁浮现 / 新目标确立（钩子不能少）
```

### 何时需要喘息章

- 连续3章高强度后
- 大高潮后1-2章内
- 世界观/设定需要解释时

### 喘息章钩子设计

虽然紧张度低，但钩子不能弱：
- 言外之意（"和你姐姐一样优秀"，但她没有姐姐）
- 承诺威胁（"今晚我会让所有人知道你做了什么"）
- 神秘物品（发现一件意义不明的东西）

## 快速应用指南

### 大纲阶段

1. 选择宏观节奏类型（递进/波浪/高原/反转）
2. 规划15个关键节点
3. 确定中点转折的具体内容

### 连载阶段（每3章）

1. 确定本单元的类型（推进/转折/情感）
2. 确保第3章有强钩子
3. 检查紧张度是否形成"低→高"曲线

### 写作阶段（每章）

1. 20%位置：进入冲突
2. 50%位置：明确转折
3. 80%位置：落下钩子
4. 交付前：五维自评
