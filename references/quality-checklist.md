# 质量检查清单

交付任何章节或整书前，先确认当前模式，再按对应清单检查。

## 0. 阶段确认

- [ ] 当前是 `策划期 / 连载期 / 收尾期`
- [ ] 写作速度：`快速模式 / 标准模式`
- [ ] 目标单章字数已锁定（标准≥3000 / 快速≥2500）
- [ ] 本次交付目标清晰：大纲 / 单章 / 连续章节 / 全书收尾

## 1. 章节基础

- [ ] **章节有明确功能**
  - 推剧情 / 推关系 / 推人物弧线 / 回收伏笔，至少命中其一

- [ ] **字数符合项目设定**
  - 轻连载：2000-3000 字
  - 标准长章：2500-4000 字
  - 重度长章：3000-5000 字

- [ ] **本章不是可删章**
  - 删除后会造成信息缺口、关系断层或节奏问题

- [ ] **时间地点清晰**
  - 场景切换时读者不会迷路

## 2. 开头检查

- [ ] **前 3 段内进入有效张力**
  - 有冲突、异常、选择、威胁、倒计时或信息反差

- [ ] **没有低效开头**
  - 不是天气描写起手
  - 不是刷牙洗脸式日常流程
  - 不是整段背景讲解

- [ ] **与上一章衔接自然**
  - 回应旧钩子，或给出清楚的时间地点切换

### 如果这是首章
- [ ] 主角已经立住，不是模糊旁观者
- [ ] 这本书的核心卖点已经显形
- [ ] 首章结尾抛出了具体主钩子

## 3. 内容推进

- [ ] **本章有不可逆变化**
  - 信息被揭示、关系被改变、局势被升级、人物做出代价性选择

- [ ] **冲突持续存在**
  - 不是只聊天、只解释、只回忆

- [ ] **有转折**
  - 至少有一次预期被改变或认知被修正

- [ ] **因果成立**
  - 事件不是靠巧合硬推

## 4. 人物与对白

- [ ] **人物行为符合既有设定**
  - 若反常，文本内给出原因

- [ ] **主要角色声音可区分**
  - 用词、语气、句长、态度存在差异

- [ ] **每段对白有功能**
  - 推动情节 / 暴露关系 / 制造冲突 / 放出信息

- [ ] **潜台词成立**
  - 角色不会把所有心事都直白说完

## 5. 叙事表达

- [ ] **展示大于讲述**
  - 少用抽象情绪总结，多用动作、细节、反应和选择

- [ ] **细节服务剧情**
  - 不是为了凑字数而堆环境描写

- [ ] **句式有变化**
  - 长短句交替，避免整段平均长度

- [ ] **段落可读性好**
  - 动作场景不堆成文字墙

## 6. AI 味清理

- [ ] **删掉空泛修饰词**
  - 如“复杂的情绪”“深邃的目光”“莫名的感觉”这类可替换空话

- [ ] **删掉机械排比和成语堆砌**
  - 不用四字词串成整段情绪

- [ ] **减少解释腔**
  - 少用“这意味着”“然而”“与此同时”“某种程度上”这类说明式连接

- [ ] **保留人物口气**
  - 角色对话不像统一的作者口吻

## 7. 悬念与连贯性

- [ ] **结尾有钩子**
  - 回应、升级或偏转至少一条悬念线

- [ ] **旧伏笔有推进**
  - 不是只开新坑不填旧坑

- [ ] **世界规则未被破坏**
  - 特别是悬疑、奇幻、科幻中的规则系统

- [ ] **人物状态已更新**
  - 位置、情绪、关系、公开目标、隐藏动机有记录

## 8. 类型特定补充

### 悬疑 / 推理
- [ ] 线索公平
- [ ] 红鲱鱼不过度作弊
- [ ] 真相能回溯自洽

### 言情 / 情感
- [ ] 关系温差有推进
- [ ] 吸引与阻碍同时存在
- [ ] 情感节点不是空喊“喜欢”

### 奇幻 / 科幻
- [ ] 规则有代价
- [ ] 世界信息是嵌入剧情而非讲义
- [ ] 新设定不会临时救场

### 动作 / 惊悚
- [ ] 空间方位清晰
- [ ] 危机层层升级
- [ ] 紧张和缓冲节奏交替

## 9. 收尾期额外检查

- [ ] 主线冲突已收束
- [ ] 主角弧线有兑现
- [ ] 关键配角去向明确
- [ ] 重大伏笔已回收或明确保留到续作
- [ ] 结局情绪与题材匹配
- [ ] 没有为了续作接口牺牲本书完整性
- [ ] 终章和尾声分工清楚，没有重复收尾

## 10. 交付前评分

| 维度 | 评分 | 说明 |
|------|------|------|
| 开头吸引力 | /10 | 前 3 段是否有效抓人 |
| 情节推进 | /10 | 本章是否真的推进主线 |
| 人物张力 | /10 | 人物关系和选择是否有戏 |
| 对话质量 | /10 | 是否自然、有功能、有声音 |
| 悬念强度 | /10 | 读者是否会想继续读 |
| 连贯性 | /10 | 与全书设定是否吻合 |
| 去 AI 味 | /10 | 是否摆脱模板腔和空话感 |
| 语言质量 | /10 | 无 AI 痕迹，用词精确？ |
| **总分** | **/80** | **>60 可交付，>70 优秀** |
