# 钩子类型与关键节点映射表

> 知道要用钩子，但不知道用什么类型？这个映射表告诉你答案。

---

## 快速查找表

| 关键节点 | 推荐钩子类型 | 悬念强度 | 目的 |
|---------|-------------|---------|------|
| **开篇钩子** | 突然揭示、震撼对话、危机时刻 | 3-4级 | 立即建立兴趣 |
| **第3章结尾** | 神秘物品、未完成的动作 | 3级 | 建立追更习惯 |
| **小高潮章** | 身份反转、两难选择 | 4级 | 情绪高点 |

## 详细映射

### 1. 开篇钩子（第1章，0%位置）

**目标**：让读者在3分钟内决定"我要追这本书"

**推荐钩子类型**（按优先级）：

#### 1A. 突然揭示（最推荐）

**示例**：
```
警官看着死者的手机，最后一条短信来自一个他认识的人——
他自己。
```

**为什么有效**：
- 立即打破预期
- 读者想知道"为什么"
- 适合悬疑、推理、惊悚

**变体**：
- 死者是主角认识的人
- 凶器上有主角的指纹
- 案发现场有主角的物品

---

#### 1B. 震撼对话

**示例**：
```
"我终于找到你了，弟弟。"
那个说着完美普通话的男人摘下面具——
露出了一张她父亲的脸。
```

**为什么有效**：
- 一句话改变一切
- 关系/身份瞬间重定义
- 适合言情、家庭、悬疑

**变体**：
- "你杀的人，其实还活着"
- "我和你母亲，二十年前就认识了"
- "那个孩子，是你的"

---

#### 1C. 危机时刻

**示例**：
```
地板开始震动，灰尘从天花板簌簌落下。
她抬头一看，裂缝正在迅速扩大——
在头顶正上方。
```

**为什么有效**：
- 立即进入紧张状态
- 读者想知道"她会不会死"
- 适合动作、灾难、冒险

**变体**：
- 被追杀的开场
- 爆炸/火灾现场
- 手术台上醒来

---

### 2. 第3章结尾（前10%，建立追更习惯）

**目标**：让读者形成"每章都有爆点"的预期

**推荐钩子类型**：

#### 2A. 神秘物品

**示例**：
```
保险箱里只有一张照片，拍摄于昨天。
照片里是熟睡中的她，从窗外角度拍摄。
```

**为什么有效**：
- 具体、可感知
- 意义不明但暗示威胁
- 适合所有题材

**变体**：
- 陌生号码的短信
- 不该存在的日记本
- 寄错的包裹

---

#### 2B. 未完成的动作

**示例**：
```
他举起枪，手指扣在扳机上——
"别动！"身后传来一个声音。
```

**为什么有效**：
- 动作被打断
- 不知道结果
- 适合动作、悬疑

**变体**：
- 开门到一半
- 告白被打断
- 即将揭露真相时被阻止

---

### 3. 小高潮章（前30%，10-30%位置）

**目标**：建立"这书有料"的信心

**推荐钩子类型**：

#### 3A. 身份反转

**示例**：
```
"你骗了我。"
"我没有——"
"你有。"她拿出那张照片，"因为这张照片里的人，
二十年前就已经死了。"
```

**为什么有效**：
- 重新定义关系
- 之前的信息全部要重新解读
- 适合悬疑、言情、权谋

---

#### 3B. 两难选择

**示例**：
```
救生艇只能载两个人。
她的丈夫和女儿都在水里，海浪越来越大。
她伸出手——
```

**为什么有效**：
- 道德困境
- 无论选什么都有代价
- 适合言情、伦理、灾难

---

### 4. 喘息章（30-70%，调节节奏）

**目标**：让读者喘口气，但仍有期待

**推荐钩子类型**（低-中强度）：

#### 4A. 言外之意

**示例**：
```
"恭喜你通过面试，"面试官笑着握住她的手，
"和你姐姐一样优秀。"

可她是独生女。
```

**为什么有效**：
- 表面正常，隐藏信息
- 需要读者解读
- 适合所有题材

**变体**：
- "你父亲生前..."（主角以为父亲还活着）
- "我们又见面了"（主角不记得见过）

---

#### 4B. 承诺/威胁

**示例**：
```
"今晚午夜之前，我会让所有人知道你十年前真正做了什么。"

匿名邮件，只有这一行字。
```

**为什么有效**：
- 明确的时间框架
- 读者知道"有事要发生"
- 适合悬疑、复仇

---

### 5. 中点转折（50%位置，全书核心）

**目标**：最大震撼，重新定义全书走向

**推荐钩子类型**：

#### 5A. 突然揭示 + 离奇消失（组合技）

**示例**：
```
"你父亲不是病死的。"律师递过来一份文件，
"他是被谋杀的。凶手是——"

灯灭了。

等电力恢复，律师不见了。
文件也不见了。
只剩下一摊血，和墙上用血写的两个字：
"别查"
```

**为什么有效**：
- 最高强度钩子
- 多个钩子叠加
- 信息+生存双重悬念

**其他组合**：
- 身份反转 + 紧急危机
- 突然揭示 + 两难选择
- 离奇消失 + 神秘物品

---

### 6. 高潮前（70-90%，加速阶段）

**目标**：迫近终点，不能停止

**推荐钩子类型**：

#### 6A. 时间限制

**示例**：
```
定时器显示 03:00。

而拆弹包里少了一根关键的红线。
```

**为什么有效**：
- 明确倒计时
- 读者感到窒息
- 适合动作、悬疑、灾难

**变体**：
- 毒药起效倒计时
- 炸弹倒计时
- 日食/月食前的仪式

---

#### 6B. 紧急危机

**示例**：
```
他推开那扇门——

她吊在房梁上，脚还在踢。
但绳子已经断了三分之一。
```

**为什么有效**：
- 立即行动需求
- 没有时间思考
- 适合动作、惊悚

---

### 7. 结局前（90-100%，终点）

**目标**：终极悬念，完美收官

**推荐钩子类型**：

#### 7A. 终极选择

**示例**：
```
两个按钮：
A. 拯救世界，但失去所有记忆（包括她）
B. 保留记忆，但世界毁灭

他的手悬在空中。
倒计时：10、9、8...
```

**为什么有效**：
- 最高代价的选择
- 主角核心价值的终极考验
- 适合所有题材的高潮

---

#### 7B. 生存悬念

**示例**：
```
他看着她掉下去。

抓住了她的手——但自己的身体也在滑向边缘。
两个人的重量，那根树枝撑不住。

他必须做出选择。
```

**为什么有效**：
- 生死一线
- 本能vs理性的冲突
- 适合动作、灾难、言情

## 钩子强度与章节位置关系

### 整体趋势

```
强度
  5 │                      ★
    │                   ╱    ╲
  4 │         ★      ╱          ★
    │      ╱   ╲  ╱                ╲
  3 │   ★        ★                      ★
    │╱    ╲    ╱   ╲                  ╱
  2 │        ★        ★            ╱
    │                  ★        ╱
  1 │                       ★ ╱
    └────────────────────────────────────→ 章节位置
      0%    25%    50%    75%    100%
           小高潮   中点   高潮前   结局
```

**原则**：
- 整体上升趋势
- 允许波动（高潮后可以有低强度钩子）
- 中点必须最高或次高
- 结局前必须最高

## 常见错误

### ❌ 开篇用低强度钩子

**错误**："他看着窗外，思考着人生。"
**后果**：读者3秒内决定弃书
**修正**：必须用3-4级强度钩子开篇

### ❌ 中点用低强度钩子

**错误**：中点只是普通章节，没有转折
**后果**：全书节奏太平，缺乏高潮
**修正**：50%位置必须有4-5级强度钩子

### ❌ 连续高强度钩子

**错误**：第8、9、10章都是5级钩子
**后果**：读者疲劳，失去冲击力
**修正**：高潮后安排喘息章（2-3级钩子）

### ❌ 钩子类型与题材不匹配

**错误**：言情用"危机时刻"，玄幻用"言外之意"
**后果**：违和感，破坏阅读体验
**修正**：参考"题材特化映射"表格

## 快速检查清单

设计章节钩子时：

- [ ] 钩子强度与章节位置匹配（对照整体趋势图）
- [ ] 钩子类型与题材匹配
- [ ] 不是"虚假悬念"（结果是误会）
- [ ] 不是"机械降神"（突然出现的解决）
- [ ] 回收前有足够铺垫（提及≥3次）
- [ ] 回收带来新的悬念或升级
