# 按题材AI味专项清单

> 不同题材有不同的AI味陷阱，针对性防治更有效。

---

## 悬疑 / 推理

### 常见AI味症状

#### 1. 线索铺陈太明显

**典型错误**：
```
请注意，这个细节将在后面成为破案关键。有经验的读者可能已经注意到...
```

**为什么错**：
- 直接告诉读者"这是线索"
- 破坏了推理的乐趣
- 像在解数学题，不是在讲故事

**改写示范**：
```
❌ "这个指纹将证明凶手是熟人。"（直接告诉）
✅ "他在门把手上看到了指纹——清晰的、没有挣扎痕迹的指纹。"

❌ "请注意，凶手就在这三个人之中。"（作者旁白）
✅ "只有三个人有钥匙。只有三个人知道密码。只有三个人——"他停住，"只有三个人。"
```

**防治要点**：
- 线索要像普通细节一样自然出现
- 让读者自己"发现"而不是被告知
- 重要线索要重复出现2-3次，但不做解释

---

#### 2. 推理过程像论文

**典型错误**：
```
第一，死者死亡时间在晚上8点到10点之间。
第二，现场没有打斗痕迹。
第三，凶手必须熟悉现场。
因此，凶手是...
```

**为什么错**：
- 一二三罗列缺乏叙事感
- 不像角色在思考，像作者在写报告

**改写示范**：
```
❌ "根据以上三点，我们可以得出结论..."
✅ "八点到十点。没有打斗。熟悉现场。"他盯着这三个事实，突然站起来，"我知道了。"

❌ "综上所述，凶手只能是..."
✅ "只有一个解释。"他转向人群，"凶手就在你们之中。"
```

**防治要点**：
- 推理要有情绪推动（愤怒、困惑、顿悟）
- 用动作打断推理（突然站起来、摔东西）
- 省略中间步骤，直接呈现结论

---

#### 3. 红鲱鱼太过刻意

**典型错误**：
```
所有人都怀疑张三。然而，真正的凶手其实是...
```

**为什么错**：
- "然而"是作者提醒，不是叙事转折
- 真正的误导应该让读者自己误判

**改写示范**：
```
❌ "张三看起来很可疑。然而，他其实是无辜的。"
✅ 写张三的可疑行为，但不解释。让读者自己怀疑。
然后在关键时刻揭示张三有不在场证明——这才是有效的误导。
```

**防治要点**：
- 不直接说"他是嫌疑人"
- 用行为和证据让读者自己怀疑
- 误导和真相要有同等强度的证据

---

### 悬疑类正向范例

**好的线索投放**：
```
他注意到死者手里的戒指——婚戒，但戴在右手。在左手有长期佩戴痕迹。
（不解释，让读者自己发现异常）
```

**好的推理时刻**：
```
"等等。"他突然停住，"那个时间，他在哪？"
没有人回答。沉默像水一样漫上来。
他知道了。
```

**好的误导**：
```
张三有动机，有机会，有凶器。
直到他们发现——那个"凶器"上，根本没有张三的指纹。
（让读者怀疑张三，然后给出反证据）
```

## 言情 / 情感

### 常见AI味症状

#### 1. 情绪形容词堆砌

**典型错误**：
```
她心中涌起甜蜜的悸动，脸颊泛起羞涩的红晕，眼神中闪烁着幸福的光芒。
```

**为什么错**：
- 连续使用"甜蜜""羞涩""幸福"等抽象词
- 像美妆广告，不像真实感受

**改写示范**：
```
❌ "她感到前所未有的幸福"
✅ "她差点撞上门框，手里的咖啡洒了一半也没在意"

❌ "他的眼中充满了爱意"
✅ "他的目光跟着她，从她拿起杯子，到放下，再到她低头时耳后的一小块皮肤"

❌ "她心中充满了悲伤"
✅ "她把那条消息看了三遍，然后长按，删除"
```

**防治要点**：
- 用身体细节代替情绪标签
- 用行为暗示心理（删除消息、反复看照片）
- 情绪的强烈程度与行为的失控程度成正比

---

#### 2. 告白场景太"完美"

**典型错误**：
```
"从第一次见到你，我就知道你是我的唯一。你的笑容如阳光般温暖，你的眼睛如星辰般璀璨...
```

**为什么错**：
- 过于诗意，脱离真实
- 真实告白通常是笨拙、紧张、词不达意的

**改写示范**：
```
❌ "我爱你。从第一次见到你开始。"
✅ "我——"他张了张嘴，又闭上，"我不是想说这个。我是说...算了。"

❌ "你是我生命中最重要的人。"
✅ "那件事，我只告诉过你。连我妈都不知道。"

❌ "我愿意为你做任何事。"
✅ "那个...你需要什么吗？我是说，不管什么都行。"
```

**防治要点**：
- 真实告白有停顿、重复、自我打断
- 用行动代替宣言（"我只告诉过你"）
- 紧张时注意力会转移到无关细节（"你扣子掉了"）

---

#### 3. 内心独白过度

**典型错误**：
```
她看着他，心里想着：他今天为什么要来？他是不是对我有意思？可是昨天他才和那个女生说话...
```

**为什么错**：
- 大段内心独白打断了叙事节奏
- 把潜台词变成了明台词，失去了暧昧

**改写示范**：
```
❌ 她心里想："他是不是喜欢我？"
✅ 他坐下时，她把手边的水杯推远了一点。又拉回来。

❌ 她疑惑："他为什么对我这么好？"
✅ 他递过来的那杯咖啡，是她喜欢的口味，不加糖。她没告诉过他。

❌ 她决定："我要忘记他。"
✅ 她删掉了那个对话框。过了五分钟，又从回收站里恢复了。
```

**防治要点**：
- 用动作暗示心理活动
- 把"她在想什么"变成"她做了什么"
- 保留潜台词，不要全说出来

---

### 言情类正向范例

**好的心动描写**：
```
他低头系鞋带。她发现自己盯着他的后脑勺看了太久，赶紧移开视线。
又忍不住看回去。
```

**好的暧昧场景**：
```
"你头发上有东西。"他伸手。
她僵住，忘了躲。
"好了。"他说，手收回去了。
她不知道那是什么东西，也不知道是不是真的有过。
```

**好的分手场景**：
```
"那就这样吧。"她说。
"好。"他说。
两个人都没动。电梯来了又走，第三趟的时候，她说："我先走了。"
"好。"
她走了。他没有。
```

## 奇幻 / 仙侠 / 科幻

### 常见AI味症状

#### 1. 设定词扎堆

**典型错误**：
```
张三运转太玄经第七层心法，催动紫府丹田中的元婴之力，施展出天罡三十六变中的第一变——身外化身。
```

**为什么错**：
- 新名词密集轰炸，读者记不住
- 像是在读游戏说明书

**改写示范**：
```
❌ "他运转太玄经第七层心法..."
✅ "他闭上眼睛。那股力量从丹田升起，像一条温热的蛇，顺着脊椎往上爬。"
（先给感觉，再慢慢解释这是什么）

❌ "这是天罡三十六变的第一变。"
✅ "一个，变成两个。两个一模一样的人，连惊讶的表情都一样。"
（先展示效果，再告诉名字）
```

**防治要点**：
- 先给感官体验，再给术语名称
- 一个新名词要有"代价"或"限制"，不能无成本使用
- 设定通过角色犯错/付出代价来展示，不是讲解

---

#### 2. 技能名称像游戏数值

**典型错误**：
```
攻击力+50，防御力+30，暴击率15%，持续3回合。
```

**为什么错**：
- 没有叙事感
- 读者无法感受到这个技能的"重量"

**改写示范**：
```
❌ "使用技能'烈焰斩'，攻击力+50"
✅ "那一刀下去，他感到手臂的骨头都在震动。刀身烫得握不住，但伤口周围的血瞬间蒸发了。"

❌ "激活护盾，防御力+30，持续3回合"
✅ "那层光罩出现的时候，他松了一口气。但很快发现，每挡下一次攻击，光罩就薄一分。三击之后，碎了。"
```

**防治要点**：
- 用身体感受代替数值
- 强调代价（"骨头震动""握不住"）
- 强调限制（"三击之后碎了"）

---

#### 3. 升级太快太顺利

**典型错误**：
```
他闭关三天，突破了筑基期，直接进入金丹期。
```

**为什么错**：
- 没有代价的升级缺乏张力
- 读者不会为"免费"的成长感到兴奋

**改写示范**：
```
❌ "他闭关三天，成功突破。"
✅ "他醒来时，发现自己动不了。丹田里有什么东西在燃烧，烧得他想要尖叫。但嗓子也烧坏了，发不出声音。"
"三天后，那团火终于灭了。他爬出洞府，发现头发全白了。"
```

**防治要点**：
- 升级必须有代价（身体、记忆、情感）
- 升级后要有"震惊众人"的爽点
- 升级不能解决所有问题，要制造新问题

---

### 奇幻类正向范例

**好的设定揭示**：
```
他试着运转那股力量。第一次，经脉像被火烧。第二次，耳朵开始流血。第三次——
他倒下了，但感觉到了。那股力量，在他的血液里，终于肯流动了。
（通过痛苦和代价来展示设定的存在）
```

**好的战斗描写**：
```
对方的剑比他快。他只能看到一道光，然后感到脖子上一凉。
不是致命伤。对方留手了。
但这一下，他知道了自己和对方的差距——不是技巧，是层次。
（用感受和认知代替技能名称）
```

**好的升级时刻**：
```
他突破的那一刻，整座山都在震动。
不是因为他的力量有多强。是因为那股力量不属于这里，被这个世界排斥。
他成功了。但他也知道了，以后每一次使用，都会是这样——与整个世界为敌。
（升级有代价，而且代价持续存在）
```

## 都市 / 职场

### 常见AI味症状

#### 1. 职场术语堆砌

**典型错误**：
```
我们需要优化KPI，提高ROI，降低成本，增强核心竞争力，实现双赢...
```

**为什么错**：
- 像年终总结，不像人在说话
- 缺乏具体的"人"的感受

**改写示范**：
```
❌ "我们需要提高ROI。"
✅ "老板看了报表，把那张纸摔在桌上。'三个月，亏了三十万。谁的责任？'"

❌ "优化资源配置"
✅ "他看着那份裁员名单。上面有他的名字——被划掉了，换成了他手下小李的。"
```

**防治要点**：
- 职场压力要具体化（裁员、 deadline、上司的怒火）
- 用人物命运承载职场概念
- 不要直接说术语，说术语带来的后果

---

#### 2. 现实压力悬浮

**典型错误**：
```
房贷、车贷、信用卡账单，压得他喘不过气。
```

**为什么错**：
- 罗列压力但无具体细节
- 读者无法感同身受

**改写示范**：
```
❌ "房贷压得他喘不过气"
✅ "每月15号，那个自动扣款的短信都会来。这个月的数字比上个月又多了三百。他看了三遍，确认自己没有看错。"

❌ "他很焦虑"
✅ "他把药瓶拿出来，又放回去。又拿出来。那瓶安眠药已经过期三个月了。"
```

**防治要点**：
- 用具体数字、具体日期、具体物品承载抽象压力
- 焦虑要有行为出口（反复检查、睡不着、吃药）
- 压力要有"最后一根稻草"的具体事件

---

#### 3. 主角开挂太假

**典型错误**：
```
关键时刻，他想起了父亲的教诲，突然开悟，成功解决了所有问题。
```
**为什么错**：
- 现实题材不应该有"顿悟解决一切"
- 读者在现实中知道这不可能

**改写示范**：
```
❌ "他想起了父亲的话，突然有了灵感"
✅ "他试遍了所有办法，都失败了。最后，他做了那件事——他父亲生前一直反对的那件事。"
"不是顿悟。是投降。"

❌ "他成功了，证明努力就会有回报"
✅ "他成功了。代价是离婚、住院、和唯一的朋友闹翻。"
"他不知道这算不算成功。"
```

**防治要点**：
- 现实题材没有"完美解决"
- 每个选择都有代价
- 结局可以是" bittersweet"，不是必须圆满

---

### 都市类正向范例

**好的职场描写**：
```
会议室里坐了十二个人。只有四把椅子是留给"自己人"的。
他坐在第五把椅子上，那把椅子比其他的矮一寸，坐久了腰疼。
他知道。这是故意的。
```

**好的压力描写**：
```
那个月的信用卡账单来了。他没有打开。
第二天，又来了。
第三天，电话来了。他看着那个号码，按了静音。
```

**好的选择描写**：
```
两个选择：留在公司，被边缘化；辞职，房贷断供。
他没有选。他拖着，直到公司替他做了选择。
那不是选择。那是逃避。但谁说逃避就不是答案呢？
```

## 跨题材通用原则

### 无论写什么题材，都要避免：

1. **总结性叙述**："总而言之""不难看出"
2. **过度解释**："这是因为...""这意味着..."
3. **抽象标签**："他很悲伤""她很开心"
4. **均匀句式**：短句短句短句，没有变化
5. **视角混乱**：一句话内切换多个角色的内心
6. **信息倾倒**：大段背景说明
7. **完美解决**：没有代价的成功

### 无论什么题材，都要追求：

1. **具体细节**：数字、物品、动作
2. **感官体验**：视觉、听觉、触觉、嗅觉
3. **行为暗示**：用动作代替心理描写
4. **句式变化**：长短交替，倒装省略
5. **视角稳定**：一节内保持同一视角
6. **信息嵌入**：通过行动揭示设定
7. **代价原则**：每个获得都有失去

---

*好故事的原则是通用的，但每个题材有自己的陷阱。针对性防治，事半功倍。*
