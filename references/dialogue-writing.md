# 对话写作规范

好对话要同时做到三件事：推动情节、暴露人物、制造张力。真正差的对白，不是不够漂亮，而是说了很多，局面却没有变化。

## 对话核心原则

### 1. 每段对白都要有任务

对白至少命中以下一项：

| 任务 | 示例 |
|------|------|
| 推动情节 | “我找到凶器了，在河边草丛里。” |
| 揭示人物 | “我不信警察，他们从来不帮我这样的人。” |
| 制造冲突 | “你骗了我。你从头到尾都在骗我。” |
| 传递信息 | “炸弹会在三点引爆。” |
| 改变关系 | “从现在起，你别再叫我名字。” |
| 制造悬念 | “你知道那天晚上真正发生了什么吗？” |

### 2. 让双方目标不同

一段对白里，最怕两个人都只是在交换信息。更有效的情况是：

- A 想套话，B 想隐瞒
- A 想靠近，B 想维持距离
- A 想求和，B 想算旧账

### 3. 真正的人不会完整表达

真实对话常包含：

- 打断
- 迟疑
- 话题转移
- 只说一半
- 用沉默代替回答

## 中文格式建议

### 标点和引号

优先使用中文全角引号和中文标点：

```text
“说话内容。”他说。
“说话内容？”她问。
“说话内容！”他大喊。
```

多行对话：

```text
“第一句话。”他说，“第二句话。”

“第一句话。
第二句话。”他说，“第三句话。”
```

对话动作：

```text
“说话内容。”他把杯子放下。
他把杯子放下。“说话内容。”
```

项目内格式要统一，不要半角全角混用。

### 标签使用

规则：

- 能辨识说话人时，省略标签
- 优先用“说”“问”这类中性标签
- 少用“愤怒地说”“悲伤地说道”这类解释式副词

| 过度使用 | 改进后 |
|---------|--------|
| “你骗了我。”他愤怒地说。 | “你骗了我。”他的声音发紧。 |
| “好的。”她高兴地同意道。 | 她眼睛亮了。“好的。” |
| “我不知道。”他悲伤地回答。 | 他低下头。“我不知道。” |

更自然的中文写法通常是：

- “对话。”他说。
- 他说：“对话。”
- “前半句，”他说，“后半句。”

### 段落划分

每个说话人的发言开始新段落。

```text
正确：
“第一句。”甲说。
“第二句。”乙回答。
“第三句。”甲点头。

错误：
“第一句。”甲说。“第二句。”乙回答。“第三句。”甲点头。
```

## 角色声音区分

每个主要角色都该有自己的声音，不要所有人都像作者换个名字在说话。

### 区分维度

| 维度 | 可区分方式 |
|------|------------|
| 用词 | 正式、俚语、专业术语、方言 |
| 句式 | 长句、短句、破碎句 |
| 语气 | 温和、尖锐、冷嘲、平淡 |
| 停顿 | 流畅、迟疑、频繁打断 |
| 习惯语 | 固定口头禅、特定说法 |

### 声音示例

教授型：

> “从理论上看，这个假设至少有三个漏洞。”

街头型：

> “扯淡。那帮人就是想蒙你。”

害羞型：

> “我……我是说，如果你不介意的话……”

傲慢型：

> “让我来告诉你，什么叫专业。”

## 潜台词

好的对白，真正的意思常常不在字面上。

| 直接说法 | 更有戏的说法 |
|---------|--------------|
| “我很生气。” | “没事。我挺好的。真的。” |
| “我喜欢你。” | “你今天看起来……不错。” |
| “我不信任你。” | “谢谢你告诉我。我会记住的。” |
| “我想离开。” | “这个地方空气不太好。” |

### 常见潜台词技巧

1. 话题转移

```text
“你爱我吗？”
“你看天气预报了吗？明天有雨。”
```

2. 反问代替回答

```text
“你杀了他吗？”
“你觉得像我这样的人会做那种事？”
```

3. 沉默和动作

```text
“你愿意原谅我吗？”
她继续翻杂志，没有抬头。
```

## 对话与动作结合

动作不是装饰，而是对白的一部分。

### 一致

> “我爱你。”她抱紧他，声音发抖。

### 矛盾

> “我完全支持你。”他看向别处，手却在口袋里攥紧。

### 动作打断

```text
“我本来想告诉你，”他停下脚步，回过身，“但我想你已经知道了。”
```

### 动作替代标签

```text
“你在撒谎。”她拍案而起。
“坐下。”他头也不抬。
```

## 常见场景的对白节奏

### 争吵场景

特征：

- 短句
- 打断
- 重复强调
- 情绪升级

示例：

> “你答应过的！”
> “情况变了！”
> “那是你的借口！”
> “你根本不懂！”
> “我当然不懂！你什么都不告诉我！”

### 告白场景

特征：

- 迟疑
- 停顿
- 寻找词语
- 真诚或尴尬

示例：

> “我……我想说……这些年，我一直在想……如果我们……”
> 她低下头，声音变小。
> “如果我们什么？”
> “如果我们早一点相遇。”

### 审讯场景

特征：

- 提问控制
- 信息不对称
- 压力递增
- 试探和误导并存

示例：

> “那天晚上你在哪里？”
> “在家。”
> “有人能证明吗？”

## 快速自查

- [ ] 这段对白里双方目标是否不同
- [ ] 说完后，局面或关系是否发生变化
- [ ] 能不能少写一句解释，多写一个动作
- [ ] 把人名遮住后，主要角色的声音还能区分吗
