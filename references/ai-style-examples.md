# AI味改写范例库

> 不是告诉用户"不要做什么"，而是展示"要怎么做"。

---

## 使用说明

1. **检测**：先用 `scripts/check_ai_style.py` 检测章节
2. **定位**：根据报告找到具体问题类型
3. **对照**：在本文件中找到对应症状的改写范例
4. **应用**：参考范例修改你的文本

---

## 症状1：空泛形容词堆砌

### 典型错误
"他感到前所未有的悲伤和绝望，内心充满了无尽的痛苦。"

### 为什么错
用抽象词总结情绪，读者无法感知。"悲伤"是什么？每个人的理解不同。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 他很悲伤 | 他盯着手机屏幕，拇指在玻璃上滑了三次，没点开那条消息 | 动作替代 |
| 她内心充满喜悦 | 她差点撞上门框，手里的咖啡洒了一半也没在意 | 行为展示 |
| 一种复杂的感觉涌上心头 | 他张了张嘴，那个名字在舌尖转了一圈，又咽了回去 | 未完成的动作 |
| 他很生气 | 他把杯子放下，杯底和桌面碰撞的声音比预想的大 | 细节放大 |
| 她感到无比孤独 | 她给自己买了生日蛋糕，插了一根蜡烛，又吹灭了 | 行为叙事 |
| 他十分紧张 | 他把那张纸对折，展开，再对折，纸边已经起毛了 | 重复动作 |
| 她非常害怕 | 她把卧室的门锁了又检查了一遍，虽然知道这没什么用 | 矛盾行为 |
| 他很失望 | 他看着那盘菜，筷子拿起又放下，最后推开了碗 | 动作链 |

### 核心技巧

**动作替代**：用具体动作代替情绪标签
- ❌ "他很生气"
- ✅ "他一拳砸在墙上，指节破皮了也没觉得疼"

**行为展示**：用行为暗示情绪
- ❌ "她很紧张"
- ✅ "她把那张纸看了七遍，还是没有填自己的名字"

**感官细节**：用感官体验代替抽象描述
- ❌ "他很痛苦"
- ✅ "他弯下腰，像是被人从里面掏空了"

## 症状2：四字成语堆砌

### 典型错误
"他情不自禁、不由自主，思绪万千、百感交集，恍然大悟、若有所思。"

### 为什么错
成语是高度浓缩的表达，连续使用会造成"概念轰炸"，失去画面感。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 他恍然大悟 | 他盯着那个数字，突然站了起来，椅子在身后倒地 | 顿悟时刻 |
| 她若有所思 | 她停下搅拌咖啡的手，勺子悬在半空，忘了放下 | 动作中断 |
| 他目不转睛 | 他连眼睛都没眨，怕一眨眼那个身影就不见了 | 恐惧失去 |
| 她忐忑不安 | 她把那个信封在手里转来转去，迟迟没拆开 | 行为暗示 |
| 他垂头丧气 | 他盯着地面，肩膀垮下来，脚步比来时重了一倍 | 身体语言 |
| 她欣喜若狂 | 她抓着那张纸在原地转了一圈，差点撞到桌角 | 动作失控 |
| 他心不在焉 | 她把那个问题问了第三遍，他才"啊"了一声 | 反应延迟 |
| 她愁眉苦脸 | 她对着镜子练习了三种笑容，最后选了最假的一种 | 行为矛盾 |

### 核心技巧

**分解动作**：把成语分解成具体动作序列
- ❌ "他夺门而出"
- ✅ "门还没完全打开，他已经挤了出去，门板撞在墙上弹回来"

**身体语言**：用身体反应代替心理成语
- ❌ "他心潮澎湃"
- ✅ "他把手按在胸口，心跳声大得像要震破耳膜"

**场景化**：把成语放进具体场景
- ❌ "她思绪万千"
- ✅ "她坐在黑暗里，想起他第一次笑的样子，最后一次说话的声音，以及那些再也不会回来的日子"

## 症状3：解释性连接词

### 典型错误
"然而他并不知道，因为这个决定，所以他将失去一切。"

### 为什么错
解释性连接词像作者在旁白，打断了沉浸感。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 然而，他不知道 | 他不知道。此刻，在城市的另一端... | 直接陈述+场景切换 |
| 因为天气不好，所以... | 雨下了三天。码头停工，他只好... | 事实陈述 |
| 虽然他很害怕，但是... | 他的手在抖。还是伸了出去 | 动作分割 |
| 这意味着... | （删除，让读者自己发现） | 信任读者 |
| 事实上... | （删除，直接陈述事实） | 直接陈述 |
| 不难看出... | （删除，直接展示） | 展示代替告诉 |
| 总而言之... | （删除，这是总结不是场景） | 场景内不要总结 |
| 某种程度上... | （删除，确定就确定，不确定就不写） | 确定立场 |

### 核心技巧

**直接陈述**：去掉连接词，直接说
- ❌ "然而，事情并没有结束"
- ✅ "事情还没有结束"

**场景切换**：用场景变化代替逻辑连接
- ❌ "他不知道的是，与此同时..."
- ✅ "他不知道。此刻..."

**动作分割**：把转折放进动作
- ❌ "虽然害怕，他还是走了过去"
- ✅ "他停了一步。然后继续走了过去"

## 症状4：时间转折词滥用

### 典型错误
"就在这时，突然门开了。下一秒，他看到了她。"

### 为什么错
"就在这时""突然""下一秒"是AI的高频词，过度使用会造成机械感。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 就在这时，门开了 | 门开了 | 直接陈述 |
| 突然，他看到了她 | 她站在门口 | 直接呈现 |
| 下一秒，电话响了 | 电话响了 | 时间自然流动 |
| 正在这时，他注意到 | 他注意到 | 删除时间标记 |
| 刹那间，他明白了 | 他明白了 | 顿悟不需要时间标记 |
| 转眼间，三天过去了 | 三天后 | 简洁时间跳跃 |
| 与此同时，在另一边... | （换行或分段）与此同时... | 排版代替连接词 |

### 核心技巧

**直接呈现**：让事件自己发生
- ❌ "突然，枪响了"
- ✅ "枪响了"

**感官先行**：用感官代替时间标记
- ❌ "就在这时，他闻到了烟味"
- ✅ "烟味。他停下脚步"

**省略时间词**：信任读者能理解时间流动
- ❌ "下一秒，他扣动了扳机"
- ✅ "他扣动了扳机"

## 症状5：情绪标签句

### 典型错误
"他感到前所未有的愤怒，心中涌起一股难以名状的情绪。"

### 为什么错
"感到...""心中..."是典型的AI情绪表达，告诉读者情绪而不是展示。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 他感到愤怒 | 他盯着那封信，指节泛白 | 身体反应 |
| 她心中涌起喜悦 | 她差点叫出声，又捂住了嘴 | 压抑的喜悦 |
| 一种复杂的感觉涌上心头 | 他想笑，又想哭，最后只是叹了口气 | 矛盾表现 |
| 他感到前所未有的孤独 | 他给自己买了生日蛋糕，插了一根蜡烛 | 行为叙事 |
| 她心中充满恐惧 | 她检查了三遍门锁，虽然知道没用 | 无效行为 |
| 他感到无比愧疚 | 他把那个名字写了又划掉，划掉又写 | 重复行为 |
| 她感到幸福 | 她走路时数着自己的脚步，七步，又七步 | 注意力转移 |
| 他感到绝望 | 他不再看表，因为知道时间已经不够了 | 放弃行为 |

### 核心技巧

**身体先行**：情绪先体现在身体
- ❌ "他很紧张"
- ✅ "他的手心全是汗，在裤子上擦了三次"

**行为矛盾**：用矛盾行为展示复杂情绪
- ❌ "她感到矛盾"
- ✅ "她拿起电话，又放下，又拿起"

**注意力转移**：情绪强烈时注意力会转移
- ❌ "他非常痛苦"
- ✅ "他数着天花板上的裂纹，第三十七道，第三十八道"

## 症状6：均匀的句式节奏

### 典型错误
"他走进了房间。他看到了她。她站了起来。他们互相看着。"

### 为什么错
均匀的短句像打字机，没有韵律感。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 他走进了房间。他看到了她。 | 他推开门，她已经在了 | 合并+省略冗余 |
| 他很生气。他说："你骗我。" | "你骗我。"声音比他想象的大 | 倒装+意外效果 |
| 他哭了。他很伤心。 | 他用手背抹眼睛，越抹越湿 | 动作代替陈述 |
| 她笑了。她很高兴。 | 她笑了一声，又笑了一声，停不下来 | 重复+失控感 |

### 句式变化技巧

**长短交替**：
```
短句。短句。短句。长句，里面有逗号，还有更多的内容，让读者喘口气。
```

**倒装句**：
```
正常：他推开了门。
倒装：门开了，被他推开。
```

**省略句**：
```
完整：他看着那个方向，一句话也说不出来。
省略：他看着那个方向。一句话。
```

**破碎句**：
```
正常：他不知道该怎么办。
破碎：不知道。怎么办。
```

## 症状7：过度书面化的对话

### 典型错误
"我认为这个假设存在三个明显的漏洞。首先...其次...最后..."

### 为什么错
真实的人不会这样说话。口语有停顿、重复、打断、半截话。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| "我认为你说的不对" | "不对。"他打断她，"全都不对" | 打断+重复 |
| "我非常感谢你帮助我" | "谢了。真的。"他顿了顿，"我是说真的" | 重复+停顿 |
| "我绝不会原谅你" | "我原——"他停住，"算了" | 自我打断 |
| "这件事让我非常困扰" | "那件事...算了，不说了" | 欲言又止 |
| "我不确定这是否正确" | "对吗？我是说...大概对吧？" | 自我怀疑 |
| "我们必须立即采取行动" | "得做点什么。现在" | 省略+紧急感 |

### 口语化技巧

**打断**：
```
"我想告诉你——"
"等一下。"她举起手，"让我先说"
```

**重复**：
```
"真的。我真的不知道。真的"
```

**半截话**：
```
"我本来以为...算了，没什么"
```

**填充词**：
```
"那个...我是说...你明白吗？"
```

**沉默**：
```
"你愿意原谅我吗？"
她继续翻杂志，没有抬头
```

## 症状8：视角混乱

### 典型错误
"他不知道她在想什么。她其实也在想同样的事。"

### 为什么错
一句话内切换两个角色的内心，读者不知道站在谁的视角。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 他不知道她在想什么。她其实在想... | 【切换到她的视角】她在想...他不知道 | 明确换行/换段 |
| 他看着她的脸，猜测她的想法 | 他看着她的脸。那张脸上没有表情 | 只写可观察的 |
| 她感到害怕，他也一样 | 【切换到他的视角】他看着她颤抖的手 | 分段处理 |

### 视角清晰技巧

**一节一人**：
```
尽可能在同一节/同一段内保持同一视角
```

**换视角标记**：
```
#### 主角视角
...

---

#### 反派视角
...
```

**只写可观察的**：
```
❌ 他不知道她在害怕
✅ 他注意到她的手在抖
```

## 症状9：信息倾倒

### 典型错误
"这是发生在2087年的未来世界，由于第三次世界大战，地球环境恶化，人类不得不迁移到地下城市。我们的主角出生于..."

### 为什么错
大段背景说明让读者失去兴趣。信息应该通过行动揭示。

### 改写示范

| 原句 | 改写 | 技巧 |
|------|------|------|
| 这个世界有严格的等级制度 | "停下。"守卫拦住他，"D级不能进A区" | 行动揭示 |
| 主角来自一个贫困的家庭 | 他数着钱包里的硬币，只有七个 | 细节暗示 |
| 这个组织很神秘 | 那个信封上没有寄件人，也没有邮票 | 物品暗示 |
| 时间紧迫 | 他看了一眼手机：23:58。还有两分钟 | 具体数字 |

### 信息揭示技巧

**行动揭示**：
```
❌ 这个世界水资源短缺
✅ 他把那半瓶水递过去，对方愣了一下才接
```

**对话揭示**：
```
❌ 主角是个侦探
✅ "你凭什么管我？" "凭这个。"他亮出警徽
```

**代价揭示**：
```
❌ 使用魔法需要代价
✅ 他念完咒文，发现自己说不出话了
```

## 综合改写示例

### 原文（重度AI味）

```
李明感到前所未有的愤怒和绝望。他情不自禁地握紧拳头，心中涌起一股难以名状的情绪。就在这时，门突然开了，他看到了张华。张华站在门口，目不转睛地看着他。

"我认为这件事我们需要好好谈谈。"张华说。

"然而，我没有什么可说的。"李明回答。

张华若有所思地点点头。事实上，他已经知道了真相。不难看出，李明在隐瞒什么。
```

### 改写后

```
李明盯着那扇门，指节泛白。门开了。

张华站在门口。他没有动，也没有说话。两个人隔着三步的距离，谁也没往前。

"我们得谈谈。"张华说。

"没什么好说的。"李明的声音比他想象的大。在空房间里，像一声枪响。

张华点点头。他已经知道了。李明的沉默，本身就是答案。
```

### 改动说明

| 原文 | 改动 | 原因 |
|------|------|------|
| 感到前所未有的愤怒和绝望 | 盯着那扇门，指节泛白 | 动作替代情绪标签 |
| 情不自禁地握紧拳头 | 删除 | 前面已暗示 |
| 就在这时，门突然开了 | 门开了 | 删除时间连接词 |
| 他看到了张华 | 张华站在门口 | 直接呈现 |
| 目不转睛地看着 | 删除 | 冗余描述 |
| 我认为这件事我们需要好好谈谈 | 我们得谈谈 | 口语化 |
| 然而，我没有什么可说的 | 没什么好说的 | 删除连接词 |
| 张华若有所思地点点头 | 张华点点头 | 删除成语 |
| 事实上，他已经知道了真相 | 他已经知道了 | 删除连接词 |
| 不难看出，李明在隐瞒什么 | 李明的沉默，本身就是答案 | 展示代替告诉 |

## 快速检查清单

修改完一段后，问自己：

- [ ] 有没有"感到/心中涌起"类情绪标签？→ 改为动作
- [ ] 有没有连续3个四字成语？→ 改为具体描写
- [ ] 有没有"就在这时/突然/然而"？→ 删除或简化
- [ ] 有没有"解释/说明/总结"？→ 改为展示
- [ ] 有没有"第一/第二/首先/其次"？→ 改为自然叙述
- [ ] 对话是否像论文？→ 增加打断、重复、半截话
- [ ] 句式是否均匀？→ 长短交替，使用倒装、省略
