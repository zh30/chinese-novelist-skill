# 情节结构模板

常见的故事结构模板，可用于规划小说章节。

## 先选故事承诺

开始拆结构前，先确认这本书卖给读者的承诺是什么：

- **悬疑承诺**：读者想知道真相
- **言情承诺**：读者想看关系升温、破裂、再靠近
- **升级承诺**：读者想看主角越来越强、越来越接近目标
- **冒险承诺**：读者想看不断打开新世界和更大危机
- **群像承诺**：读者想看不同人物线最终交汇

结构必须服务承诺，不能只像一个结构。

## 三幕式结构

最经典，适用于大多数短篇和中篇。

### 第一幕：设置，约 25%

| 章节 | 内容 |
|------|------|
| 第1章 | 现状、人物、缺口 |
| 第2章 | 激励事件，打破现状 |
| 第3章 | 拒绝、犹豫，最终决定行动 |

### 第二幕：对抗，约 50%

| 章节 | 内容 |
|------|------|
| 第4-5章 | 进入新局面，开始遭遇挑战 |
| 第6章 | 中点，重大转折或重大认知变化 |
| 第7-8章 | 困难升级，代价抬高 |

### 第三幕：结局，约 25%

| 章节 | 内容 |
|------|------|
| 第9章 | 最低点，似乎一切失败 |
| 第10章 | 高潮，对决或最终选择 |
| 终章 | 余波与新常态 |

### 放到长篇时怎么拉长

- 第一幕不要拖太久，尽量在前 10-20% 进入主线
- 第二幕按 `小目标 -> 小胜利 -> 更大代价` 循环推进
- 中点必须改变读者对局势的理解
- 第三幕前要让主角付出真正代价，不然高潮会轻

## 英雄之旅

适合冒险、奇幻、成长型故事。

| 阶段 | 章节 | 内容 |
|------|------|------|
| 平凡世界 | 第1章 | 主角的旧秩序 |
| 冒险召唤 | 第1-2章 | 激励事件 |
| 拒绝召唤 | 第2章 | 犹豫与抗拒 |
| 遇见导师 | 第2-3章 | 获得指引 |
| 跨越门槛 | 第3章 | 进入新世界 |
| 考验盟友敌人 | 第4-5章 | 新规则与新关系 |
| 接近洞穴 | 第5-6章 | 逼近核心威胁 |
| 苦难煎熬 | 第6-7章 | 重大失败或接近毁灭 |
| 奖赏 | 第7-8章 | 获得关键力量或信息 |
| 返回之路 | 第8章 | 回归途中再遭阻碍 |
| 复活 | 第9章 | 最终考验与蜕变 |
| 满载而归 | 第10章 | 带着改变回归 |

## 悬疑小说结构

适合侦探、推理、惊悚题材。

### 第一幕：谜题出现

| 章节 | 内容 | 钩子 |
|------|------|------|
| 第1章 | 案件发生 / 尸体出现 | 谁干的 |
| 第2章 | 主角接手，发现案件特殊 | 为什么特殊 |
| 第3章 | 初查，第一批线索浮现 | 线索指向谁 |

### 第二幕：调查深入

| 章节 | 内容 | 钩子 |
|------|------|------|
| 第4章 | 审讯与排查 | 谁在撒谎 |
| 第5章 | 意外线索出现 | 我们是不是一开始就错了 |
| 第6章 | 第二起事件或旧案连接 | 这是同一只手吗 |
| 第7章 | 主角被卷入危险 | 主角会不会成为下一个目标 |

### 第三幕：真相揭示

| 章节 | 内容 | 钩子 |
|------|------|------|
| 第8章 | 重大突破或反转 | 我们信任的人有问题 |
| 第9章 | 最终逼近真相 | 真相是什么，代价是什么 |
| 第10章 | 真相落地与余波 | 真的结束了吗 |

## 言情小说结构

适合爱情和关系推进题材。

### 第一幕：相遇

| 章节 | 内容 | 情感节点 |
|------|------|---------|
| 第1章 | 主角 A 的现状和缺口 | 读者共情 A |
| 第2章 | 主角 B 的现状和缺口 | 读者共情 B |
| 第3章 | 相遇，第一印象不好或错位 | 有吸引，也有阻碍 |

### 第二幕：发展

| 章节 | 内容 | 情感节点 |
|------|------|---------|
| 第4-5章 | 被迫靠近，逐渐看见对方 | 关系升温 |
| 第6章 | 高亲密时刻 | 接近 |
| 第7-8章 | 误解、秘密、外力阻碍 | 拉远 |
| 第9章 | 关系断裂或看似无望 | 情绪谷底 |

### 第三幕：和解

| 章节 | 内容 | 情感节点 |
|------|------|---------|
| 第10章 | 克服障碍，完成选择 | 高潮 |
| 终章 | 新关系状态 | 圆满或留白 |

### 言情特别提醒

- 吸引和阻碍必须同时存在
- 甜和虐要交替，不要连续重复同一种拉扯
- 真正推进关系的，是选择和行动，不是反复口头确认喜欢

## 惊悚 / 动作结构

适合快节奏、强压迫感题材。

```text
第1章：危机出现
第2章：应对计划
第3章：追逐 / 对抗
第4章：逆转机会
第5章：最终对抗
第6章：结局余波
```

特点：

- 节奏快
- 高密度事件
- 时间压力持续存在
- 悬念强度递增

## 反转结构

适合心理惊悚和反转型悬疑。

```text
第1-2章：建立“真实”情况
第3-4章：出现疑点，但不明显
第5-6章：第一次小反转
第7-8章：第二次反转
第9-10章：最终反转
```

关键：

- 前期埋下看似无辜的线索
- 每次反转都合乎逻辑
- 避免机械降神

## 多线叙事结构

适合多主角和复杂剧情。

### 交叉剪辑模式

```text
第1章：主角 A
第2章：主角 B
第3章：主角 A 推进
第4章：主角 B 推进
第5章：线索交汇
```

### 时间线模式

```text
第1章：现在
第2章：过去
第3章：现在 +1 天
第4章：过去 +1 月
```

收敛原则：

- 各线最终必须交汇
- 早期看似无关，后期必须有关
- 收敛时要产生原来如此的效果

## 中文网文 / 长篇连载结构

适合 20-100 章以上的持续推进型作品。

### 核心节奏

```text
起钩 -> 给目标 -> 遇阻 -> 小赢 -> 反噬 -> 再升级 -> 阶段回收 -> 更大目标
```

### 分卷思路

| 阶段 | 常见章数占比 | 任务 |
|------|-------------|------|
| 开卷 | 前 10-15% | 立人设、立承诺、立主钩子 |
| 第一阶段推进 | 20-25% | 建立规则，给第一次明显成果 |
| 中段升级 | 25-35% | 扩大舞台，抬高代价，关系重组 |
| 大转折 | 10-15% | 认知翻转或重大失败 |
| 终局推进 | 15-20% | 回收主线，处理终极对抗 |

### 连载章注意点

- 每 3-5 章最好出现一次明显兑现
- 每个小阶段要有局部闭环
- 追更节点的章末，优先落在新危机、身份反转、重大发现、关系突变
- 不要连续多章都只做过渡

## 单章内部结构

### 标准模板

```text
开头（10%）：钩子 + 上下文连接
发展（60%）：事件推进 + 冲突
高潮（20%）：本章最高点
结尾（10%）：悬念钩子 + 下章铺垫
```

### 更适合连载的模板

```text
进入（15%）：异常 / 威胁 / 任务
施压（35%）：目标受阻，信息不对称
转折（25%）：认知变化 / 关系变化 / 局面恶化
余波（15%）：人物反应与决定
钩子（10%）：把问题抛给下一章
```

### 无效章节结构

```text
开头：漫长铺垫 / 背景说明
中间：日常活动 / 对话流水账
结尾：平淡结束 / 无悬念
```

## 结构选择指南

| 故事类型 | 推荐结构 | 章节数 |
|---------|---------|--------|
| 冒险 / 奇幻 | 英雄之旅 | 8-12章 |
| 侦探 / 悬疑 | 悬疑结构 | 8-10章 |
| 言情 | 言情结构 | 6-10章 |
| 动作 / 惊悚 | 惊悚结构 | 5-8章 |
| 心理 / 反转 | 反转结构 | 6-8章 |
| 多主角 | 多线叙事 | 10-15章 |
| 微型小说 | 3章结构 | 3章 |
| 中文网文 / 长篇连载 | 分卷推进结构 | 20章以上 |
