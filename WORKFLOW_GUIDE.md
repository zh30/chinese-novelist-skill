         ▼

## 悬念管理流程图

### 悬念生命周期

```
┌──────────────────────────────────────────────┐
│              悬念生命周期状态机                 │
├──────────────────────────────────────────────┤
│                                              │
│  ⚪ 待激活 ──→ 🟢 活跃 ──→ ✅ 已回收        │
│              │    │                         │
│              │    ├─→ 🟡 即将过期           │
│              │    │      （5章未提及）        │
│              │    │                         │
│              │    └─→ 🔴 已过期             │
│              │           （10章未提及）       │
│              │                               │
│              └────────→ 🚫 已废弃            │
│                                              │
└──────────────────────────────────────────────┘

关键规则：
- 活跃悬念上限：6个
- 每章至少提及1个已有悬念
- 每3-5章推进至少1个悬念
```

每章写作前：

         │
         ▼

### 宏观层（全本）

```
紧张度
  5 │                               ★
    │                          ╱    ╲
  4 │             ★         ╱          ★
    │          ╱   ╲    ╱                  ╲
  3 │    ★  ╱       ★                         ★
    │  ╱   ╲    ╱    ╲                     ╱
  2 │       ★╱         ★                ╱
    │                   ★            ╱
  1 │                      ★    ╱
    └────────────────────────────────────→
      0%   10%   25%   50%   75%   90%  100%
      │      │      │      │      │      │
      │      │      │      │      │      └─ 结局
      │      │      │      │      └─ 高潮前
      │      │      │      └─ 高潮
      │      │      └─ 中点转折
      │      └─ 小高潮
      └─ 开篇钩子

四幕结构：
[第一幕 25%] [第二幕 50%] [第三幕 20%] [第四幕 5%]
```

### 中观层（章节组）

```
黄金三章模式：

第N章        第N+1章      第N+2章
[铺垫]  →  [升级]   →  [小高潮+钩子]
  │            │            │
建立情境    矛盾激化    情绪高点+新问题
紧张度：★★☆  →  ★★★   →  ★★★☆

每3章一个节奏单元
单元结束后必须推进主线
```

### 微观层（单章）

```
单章内部结构：

[0-20%]    [20-50%]    [50-80%]    [80-100%]
  ↓           ↓            ↓            ↓
进入冲突   矛盾激化    转折/升级    钩子落下
紧张度：    紧张度：     紧张度：     紧张度：
★★☆       ★★★        ★★★☆       ★★★★☆

每章至少3次情绪变化
字数：≥3000字
```
